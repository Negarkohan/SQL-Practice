# Intermediate SQL (AdventureWorks2019)

This folder contains intermediate-level SQL scripts for the **AdventureWorks2019** database in SQL Server.

## About
- The scripts cover intermediate SQL topics such as joins, aggregations, and subqueries.  
- Each `.sql` file includes comments in English and example usage.  
- More intermediate-level topics will be added over time.  

## How to Use
1. Open SQL Server Management Studio (SSMS).  
2. Connect to your instance with the **AdventureWorks2019** database.  
3. Run the scripts in this folder.  
4. Follow the comments for explanations and usage examples.
