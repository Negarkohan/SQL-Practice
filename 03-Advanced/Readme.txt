# Advanced SQL (AdventureWorks2019)

This folder contains advanced SQL functions created for the **AdventureWorks2019** database in Microsoft SQL Server.

## About
- Each `.sql` file contains one or more SQL functions.  
- Functions are well-documented with inline comments in English.  
- Example usage of each function is included inside the SQL file.  

## Topics Covered
- Scalar functions  
- Inline table-valued functions (TVF)  
- Multi-statement table-valued functions (TVF)  

## How to Use
1. Open SQL Server Management Studio (SSMS).  
2. Connect to your instance with the **AdventureWorks2019** database.  
3. Run the SQL scripts from this folder.  
4. Call the functions using the examples inside each file.
